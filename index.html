Política de Privacidad de Kai Events

Última actualización: julio de 2026

Kai Events es una aplicación de punto de venta de uso interno, desarrollada por KaiSoft para la gestión de ventas, clientes y productos del negocio que la opera. No está dirigida al público general ni a menores de edad, y su acceso está restringido al personal autorizado del negocio.

1. Datos que recopila la aplicación

DatoPara qué se usa
Usuario y contraseñaAutenticar al personal que usa la aplicación. La contraseña se almacena únicamente como un hash (SHA-256), nunca en texto plano.
Datos de clientes (nombre, teléfono, correo, RNC/cédula)Registrados por el propio negocio para emitir ventas y comprobantes.
Datos de productos y ventasOperación normal del punto de venta: inventario, transacciones, reportes.
Fotos (galería)Únicamente para que el negocio seleccione el logo que se muestra en la aplicación y en los tickets impresos.

La aplicación no recopila datos de ubicación. El permiso de Bluetooth se usa exclusivamente para conectar impresoras térmicas y está declarado ante Google Play como "nunca para ubicación" (neverForLocation).

2. Uso de la cámara

La cámara se utiliza únicamente para escanear códigos de barras de productos dentro del flujo de ventas. La aplicación no captura, almacena ni transmite fotografías o video de la cámara.

3. Con quién se comparten los datos

Los datos se sincronizan exclusivamente con el servidor propio del negocio que opera la aplicación (configurado por el administrador en Ajustes). No se comparten con terceros, redes de publicidad, ni se utilizan con fines comerciales ajenos al negocio.

4. Almacenamiento y seguridad

  • Los datos se almacenan localmente en el dispositivo (base de datos SQLite) para permitir el funcionamiento sin conexión, y se sincronizan con el servidor del negocio cuando hay conexión disponible.
  • Las contraseñas se protegen mediante hash SHA-256.
  • La comunicación con el servidor usa HTTPS cuando el negocio configura su servidor con certificado TLS.

5. Retención y eliminación de datos

Los datos se conservan mientras el negocio que administra la aplicación lo considere necesario para su operación. Para solicitar la eliminación de datos personales almacenados, contacta al correo indicado abajo.

6. Permisos declarados en Google Play

  • Cámara: escaneo de códigos de barras.
  • Bluetooth: conexión con impresoras térmicas (nunca para ubicación).
  • Almacenamiento/Fotos: selección del logo de la empresa.
  • Internet: sincronización con el servidor del negocio.

7. Cambios a esta política

Esta política puede actualizarse para reflejar cambios en la aplicación. La fecha de la última actualización se indica al inicio de esta página.
